My So-Called Civic Life

My So-Called Civic Life is an application that allows users to find their local, state, and federal public servants, and keep track of their legislative activity.

I used Vue.js, a JavaScript framework, as well as PostgreSQL and Django. The application has full CRUD (create, read, update, delete) functionality. Real-time data is provided by external API calls utilizing the Google Civic Information API and the ProPublica Congress API.
